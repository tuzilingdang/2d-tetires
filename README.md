# 2d版俄罗斯方块

## 游戏介绍
作为经典游戏之一，游戏机中的俄罗斯方块是童年的回忆，本游戏基于Vue，Vuex，vuecli实现了web版俄罗斯方块

### 本地运行
下载源代码到本地，进入项目根路径，执行

```
npm run serve
```
打开 localhost://8080可以运行游戏

### 压缩打包
进入项目根路径，执行
```
npm run build
```
